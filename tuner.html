What is Pro Tuner

Pro Tuner is a well designed Chromatic Tuner on iPhone/iPad especially for Guitar and Bass player.

It's highly accurate and has 91 build-in tuning modes. Moreover, you can play REAL guitar reference sounds by tapping the strings.

Pro Tuner supports two input methods: build-in microphone and line-in . Pro Tuner supports Guitar Clip Microphone, which especially makes tuning highly accurate.

It's Accurate

Pro Tuner use highly accurate algorithm to make sure your guitar sound more accurate then ever, even by using iPhone inner-microphone.

It's Pro

Pro Tuner has 91 tuning modes build inside. Each node has real guitar reference sound. Pro Tuner is light, but PRO!

It's Universal

You don't need purchase iPhone and iPad separately. Pro Tuner is universal APP. It's works on all of your iOS devices.

Support Clip Microphone

With a clip microphone, you tuning will become more accurate.

Features List

  • User-friendly. Extremely easy to use
  • Universal APP. Purchase once, available on all of your iPhone/iPod/iPad.
  • Support chromatic tuning. Especially optimized for acoustic/electric Guitar and Bass plus 7-string guitars, 6-string basses and more.
  • Reference Sounds. Each string has real guitar reference sound.
  • Tuning Mode. 91 builds inside.
  • Input-Level indicator. Make sure your input level always be right.
