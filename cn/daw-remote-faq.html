使用时遇到了问题?

如果你在使用DAW Remote时遇到问题, 请在下面FAQ中寻找答案.
如果想对DAW Remote有意见, 请到DAW Remote页面提出

Q我如何才能让DAW Remote连接上我的MAC电脑?

请按照如下指示操作:

  • 请先确保DAW Remote和你的MAC电脑在同一个WIFI下. 因为数据的传输是依赖网络连接的

  • 在MAC中运行 “Audio MIDI Setup”.

  • 选择 “Show MIDI Window”.

  • 在窗口中, 双击 “Network” 打开 “MIDI Network Setup” 面板.

  • 在 “MIDI Network Setup” 中, 点击 + “My Sessions”. 如果已经有了Session, 请跳过此步骤.

  • 任意选择一个Local name和一个Bonjour name. Local name 会将在你的电脑软件中显示, Bonjour name 会在你的iPhone中显示.

  • 确保 “Who may connect to me” 设置为了 “Anyone”.

  • 打开DAW Remote. 若一切正常,你已经能看到你的电脑出现在APP中了.

Q我如何在Apple Logic中设置DAW Remote

  • 运行 Apple Logic, 选择 Logic Pro -> Preferences -> Control Surfaces -> Setup...

  • 在 Setup 窗口中, 选择 New -> Install...

  • 在 Install 窗口中, 找到并添加 TranzPort

  • 回到 Setup 窗口, 选择 TranzPort Input 和 Out Port 为你之前在MIDI Network Setup面板中设置的local name.

Q我总是遇到 "Connection Lost" 的问题

请确保在 Network MIDI Setup 面板中 Who may connect to me 设置为了 Anyone.

Q我按帮助中设置, 但是依然会碰到 "Connection Lost" 的问题

请尝试关闭你的无线路由器中的防火墙再试.(因为某些国产路由器会自以为是地吞掉他们认为未知的协议.)

若问题依旧, 你可以试着创建一个Computer-to-Computer 的网络. 以下是步骤.

  • 确保 MIDI Network Session 已经打开.

  • 打开你MAC电脑的 WI-FI, 并且选择 Create Network...

  • 给网络选个名字. 打开iPhone,连接到刚刚创建的那个WIFI网络中. 运行 DAW Remote并再试一次.
